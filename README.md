
# 📍 BUPT 校园导航系统 (Campus Navigation System)

本项目为《数据结构》课程设计作业一（图结构应用）。基于 Python 构建了完整的校园路网拓扑模型，底层采用图数据结构，实现了基于 Dijkstra 算法的最短路径规划、图的深度/广度优先遍历以及路网数据的动态管理。

---

## 🚀 核心功能 (对应课程设计要求)

本项目严格按照《数据结构课程设计》PPT 要求实现，具体功能如下：

1. **校园地图图论建模 (Graph Representation)**
   - 使用带权无向图表示校园建筑与道路。顶点 (Vertex) 代表校园建筑及出入口，边 (Edge) 代表道路，权值 (Weight) 为实际物理距离。
   - 底层采用**邻接表 (Adjacency List)** `{Node_ID: {Neighbor_ID: Distance}}` 进行物理存储与内存加载。
2. **智能路径查询 (Shortest Path)**
   - 支持查询任意两点（建筑名或节点）之间的最短路径与距离。
   - **算法实现**：采用基于最小堆优化的 **Dijkstra 算法**，处理多入口建筑的“单源多汇”最短路径寻优。
   - **可视化输出**：在终端输出详细的文字路径描述，并通过 `matplotlib` 在简易地图上实时高亮绘制导航路线。
3. **图的遍历 (Graph Traversal)**
   - 支持从任意建筑节点出发，对整个校园图结构进行遍历。
   - 包含**广度优先搜索 (BFS)** 与 **深度优先搜索 (DFS)** 两种模式，并输出遍历的前置节点序列。
4. **路网数据动态管理 (Dynamic Modification)**
   - 支持在程序运行期间通过交互菜单**添加**或**删除**建筑节点与道路。
   - 支持修改结果的持久化保存（覆写 CSV 数据集），确保增删改操作在重启后依然生效。

## 💻 运行与使用指南

1. 确保已安装所需的 Python 环境及第三方库。
2. 在终端运行主程序：
   ```bash
   python map_system.py
   ```
3. 程序将启动基于控制台的交互式主菜单，根据提示输入功能序号 (1-5)：
   - **[1] 查询最短路径**：输入起点与终点名称，系统将计算距离、打印文字引导，并弹出窗口展示路线图。关闭图片窗口后可继续其他操作。
   - **[2] 遍历校园建筑**：输入起点名称，选择 BFS 或 DFS，系统将打印探索到的节点列表。
   - **[3/4/5] 数据管理**：根据提示输入坐标和 ID 添加/删除图的节点与边，完成后输入 `5` 将内存中的图结构回写至 CSV 文件。

## 📊 算法复杂度摘要 (拓展补充)

为了便于撰写实验报告，此项目的核心算法复杂度如下：
- **最短路径 (Dijkstra + Heapq)**: 
  - 时间复杂度: $O(E \log V)$ （$E$ 为边数，$V$ 为顶点数）。使用优先队列极大优化了寻路效率。
  - 空间复杂度: $O(V)$ 用于存储距离表和优先队列。
- **图的遍历 (DFS / BFS)**: 
  - 时间复杂度: $O(V + E)$。每个顶点和每条边最多被访问一次。
  - 空间复杂度: $O(V)$，用于维护 `visited` 集合以及辅助队列 (BFS) 或递归栈 (DFS)。
